## 설계 원칙

### 1. 사용자 중심 설계
- **최소 클릭**: 주요 작업을 3클릭 이내에 완료
- **진행 상태 표시**: 측정 → 계산 → 결과의 단계별 진행률 표시
- **실시간 피드백**: 입력 즉시 유효성 검사 및 미리보기

### 2. 데이터 입력 최적화
- **테이블 형태 입력**: Excel과 유사한 그리드 입력 방식
- **키보드 네비게이션**: Tab, Enter, 방향키로 셀 이동
- **자동 계산 미리보기**: 입력 시 실시간 유량 추정값 표시
- **Excel 붙여넣기 지원**: 기존 데이터 복사/붙여넣기

### 3. 오류 방지
- **필수 입력 강조**: 빨간 별표(*) 표시
- **범위 검증**: 수심, 유속 등의 합리적 범위 검사
- **경고 메시지**: 이상값 입력 시 경고 (삭제 X, 확인만)

### 4. 접근성
- **고대비 색상**: 시인성 높은 색상 조합
- **큰 터치 영역**: 모바일에서도 쉬운 터치
- **단축키 지원**: 주요 기능에 키보드 단축키

### 버튼 스타일

| 유형 | 용도 | 스타일 |
|------|------|--------|
| Primary | 주요 액션 (저장, 계산) | 파란색 배경, 흰색 텍스트 |
| Secondary | 보조 액션 (취소, 이전) | 흰색 배경, 회색 테두리 |
| Danger | 삭제, 경고 | 빨간색 배경 |
| Ghost | 텍스트 링크 | 투명 배경, 파란색 텍스트 |

### 입력 필드

```
┌─────────────────────────────────────┐
│ 레이블 *                            │
│ ┌─────────────────────────────────┐ │
│ │ placeholder text                │ │
│ └─────────────────────────────────┘ │
│ 도움말 텍스트                        │
└─────────────────────────────────────┘
```

### 데이터 테이블

- **헤더**: 회색 배경, 굵은 글씨
- **행**: 흰색 배경, 호버 시 하늘색
- **선택**: 파란색 테두리
- **편집 모드**: 노란색 배경

## HTMX 인터랙션 패턴

### 1. 실시간 유효성 검사

```html
<input
    name="depth"
    hx-post="/api/validate/depth/"
    hx-trigger="blur"
    hx-target="#depth-error"
    hx-swap="innerHTML"
/>
<span id="depth-error"></span>
```

### 2. 실시간 계산 미리보기

```html
<form
    hx-post="/api/preview/discharge/"
    hx-trigger="input delay:500ms"
    hx-target="#discharge-preview"
>
    <!-- 입력 필드들 -->
</form>
<div id="discharge-preview">
    예상 유량: -- ㎥/s
</div>
```

### 3. 측선 추가 (동적 행)

```html
<button
    hx-post="/measurement/add-row/"
    hx-target="#data-table tbody"
    hx-swap="beforeend"
>
    + 측선 추가
</button>
```

### 4. 자동 저장

```html
<form
    hx-post="/measurement/autosave/"
    hx-trigger="input delay:2000ms"
    hx-indicator="#saving-indicator"
>
    <!-- 폼 내용 -->
</form>
<span id="saving-indicator" class="htmx-indicator">
    저장 중...
</span>
```
