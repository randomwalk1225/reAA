# 박승혁 제안 종합 분석

## 지표수-지하수 연계 스마트 수문조사 앱 구상

---

## 1. 배경 및 문제 인식

박승혁님은 현재 수문조사 업무에서 다음과 같은 한계점을 인식하고 있습니다:

### 현재 상황
- 유량조사 후 H-Q Rating Curve를 수작업으로 도출
- 연도별, 수문기간별, 관측수위별로 각각 분리하여 분석
- 한강홍수통제소 등 과거 자료를 토대로 최종보고서 작성
- **유역 내 관측지점이 없는 경우** 신규로 자료를 축적해야 하는 문제

### 개선 필요성
> "실시간으로 조사지점마다 앱상에서 보여줄 수 있다면 좀 더 활용가능성이 클 듯요."

## 2. 제안하는 시스템 아키텍처

박승혁님이 구상하는 시스템은 **3단계 분석 파이프라인**으로 구성됩니다:

```
┌─────────────────────────────────────────────────────────────────┐
│                    스마트 수문조사 앱                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [1단계] 실시간 데이터 수집                                        │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  자동계측 수위정보 → 유량 시계열 자료 생성                    │   │
│  │  (H-Q Rating Curve: Q = a(H+b)^c)                        │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              ↓                                  │
│  [2단계] 기저유출분석 (1년 이상 데이터 축적 후)                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Eckhardt Recursive Digital Filter                       │   │
│  │  - Alpha (α): 필터 파라미터                                │   │
│  │  - Beta (β): 대수층/하천 특성 반영                          │   │
│  │    ├ β=0.25 → 암반대수층, 간헐하천                         │   │
│  │    ├ β=0.50 → 충적대수층, 간헐하천                         │   │
│  │    └ β=0.80 → 충적대수층, 상시하천                         │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              ↓                                  │
│  [3단계] 통합 모델링 검증                                         │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Integrated Surface-Subsurface Water Hydrologic Modeling │   │
│  │  결과와 비교 평가 → 신뢰도 검증                             │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              ↓                                  │
│  [최종 결과]                                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  지표수-지하수 연계 분석 결과                               │   │
│  │  - 지하수 함양량 산정                                      │   │
│  │  - 대수층/하천 유형 자동 평가                               │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 핵심 기술 요소

### 3.1 H-Q Rating Curve (수위-유량 관계곡선)

**목적**: 연속 측정되는 수위(H)로부터 유량(Q)을 산출

**일반 공식**:
```
Q = a(H + b)^c
```
- Q: 유량 (cms, ㎥/sec)
- H: 상대수위 (실수위 + 조정수위)
- a, b, c: 매개변수 (하천 특성에 따라 결정)

**실제 사례** (보은내북_괴산청천):
```
Q = 3.22 × (h - 0.001)^1.941  (조건: 0.24 < h ≤ 0.5)
```

### 3.2 Eckhardt 기저유출 분리법

**참고문헌**:
- Eckhardt, K. 2012. "Technical note: Analytical sensitivity analysis of two parameter recursive digital baseflow separation filter." *Hydrology and Earth System Sciences* 16: 451-455.

**핵심 공식**:
```
B_{k+1} = ((1-BFI_max) × α × B_k + (1-α) × BFI_max × Q_{k+1}) / (1 - α × BFI_max)
```

**파라미터 의미**:
| 파라미터 | 의미 | 조정 방법 |
|---------|------|----------|
| α (Alpha) | 디지털 필터링 매개변수 (감쇠상수) | 가뭄기간 하천유량 자료로 추정 |
| β (BFI_max) | 기저유출 비율 최대값 | 대수층/하천 특성에 따라 선택 |

**Beta 값 선택 기준**:
| β 값 | 대수층 유형 | 하천 유형 | 특징 |
|------|-----------|----------|------|
| 0.25 | 암반대수층 | 간헐하천 | 투수성 낮음, 일시적 흐름 |
| 0.50 | 충적대수층 | 간헐하천 | 투수성 높음, 일시적 흐름 |
| 0.80 | 충적대수층 | 상시하천 | 투수성 높음, 항상 흐름 |

### 3.3 통합 수문모델링 검증

**개념**: 기저유출분석 결과를 **Integrated Surface-Subsurface Water Hydrologic Modeling** 결과와 비교하여 상호 검증

**검증 프로세스**:
```
기저유출분석 결과 ←─비교 평가─→ 통합 수문모델링 결과
       │                              │
       └────────────┬────────────────┘
                    ↓
            신뢰도 검증 완료
```

## 5. 기대 효과

박승혁님이 제안하는 시스템이 구현되면:

1. **실시간 모니터링**: 조사지점마다 앱에서 바로 데이터 확인 가능
2. **자동화**: 수작업으로 하던 Rating Curve 분석이 자동화
3. **표준화**: Eckhardt 필터를 통한 일관된 기저유출분석
4. **검증 가능**: 두 가지 독립적 방법론으로 상호 검증
5. **확장성**: 유역 내 관측지점 없어도 집수유역면적비로 추정 가능

## 7. 관련 도구 및 기술 스택

### 전세계 대표적 통합 수문모델링 도구

| 도구명 | 설명 | 비고 |
|--------|------|------|
| **Hyrosphere** | 통합 지표수-지하수 수문모델링 | 전세계 대표 도구 |
| **SWAT-MODFLOW** | SWAT + MODFLOW 결합 모델 | 전세계 대표 도구 |
| **DWAT** | UN 개발, 전세계 배포 | 국제 표준 도구 |
| **SWAT-K** | 한국형 SWAT 모델 | 국책연구원 개발 (10년, 100억원) |

### 박승혁님 기술 배경
- SWAT-K를 현장에서 실제 working modeling에 활용한 경험
- 전문가가 되기 위해 꾸준히 연구해온 분야
- SCI 수준 (WRR, WATER 저널) 논문 게재 가능한 기술적 깊이

### 활용 가능 분야
- 국내 수자원 관리
- **사막 지역 지하수 탐사** (물 찾기)
- 물 부족 지역 수자원 관리

## 8. 결론

박승혁님은 **"지표수-지하수 연계 스마트 수문조사 앱"**을 통해:

1. **수위 측정** → **유량 계산** (H-Q Rating Curve)
2. **유량 시계열** → **기저유출 분리** (Eckhardt 필터)
3. **기저유출 분석** → **통합 모델링 검증** (Surface-Subsurface Modeling)

이 세 단계를 하나의 플랫폼에서 실시간으로 수행할 수 있는 시스템을 구상하고 있습니다.

이는 단순한 데이터 수집 앱이 아니라, **지하수 함양량 산정**과 **수자원 관리**를 위한 **과학적 분석 도구**입니다.
